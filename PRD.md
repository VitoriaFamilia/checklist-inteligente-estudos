# Mini PRD - Checklist Inteligente de Estudos

## 1. O Problema
Estudantes que conciliam trabalho (8h às 17h) e estudos (até as 22h) enfrentam exaustão e dificuldade em priorizar tarefas urgentes de diferentes instituições (UBEC, FAETEC, PDA).

## 2. Solução Proposta
Uma aplicação de checklist com sinalização visual de urgência. O sistema destaca automaticamente em vermelho tarefas que vencem em menos de 48 horas.

## 3. Requisitos Funcionais (MVP)
- Listagem de tarefas por instituição.
- Filtro de prioridade (Alta, Média, Baixa).
- Persistência de dados via API (JSON Server).

## 4. O que faria diferente (Próximos Passos)
- Implementação de autenticação de usuário.
- Notificações via navegador para prazos expirando.
