# Rotina Organizada - Checklist de Estudos

Este projeto é um gerenciador de tarefas Full-Stack focado em estudantes com alta carga horária.

## 🔗 Links Úteis
- **Deploy do Projeto:** [Clique aqui](https://checklist-inteligente-estudos.vercel.app/)
- **Documentação de Produto (Mini PRD):** [Ver PRD Completo](./PRD.md)

## 🛠️ Tecnologias
- **React + TypeScript**
- **Axios** (Consumo de API)
- **CSS Modules** (Estilização)
- **JSON Server** (Back-end no Render)

## 🚀 Como rodar
1. `npm install`
2. `npm run dev`
